# Projet Jenkins + Terraform + Ansible

## 📋 Description
Automatisation du déploiement d'un serveur web Apache sur AWS EC2.

## 🛠️ Technologies utilisées
- Jenkins (CI/CD)
- Terraform (Infrastructure as Code)
- Ansible (Configuration Management)
- AWS EC2

## 🚀 Pipeline
1. Terraform crée une instance EC2
2. Ansible installe et configure Apache
3. Test de validation
